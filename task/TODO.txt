Тестовое задание достаточно простое. Сделать приложение отправляющее пост запросы по адресу http://httpbin.org/post и складывающее ответы списком ниже формы.

Сама форма может иметь неограниченное кол-во полей и кнопочку для добавление нового либо не заполненное нижнее поле, при клике по которому добавляется еще одно (пример в аттаче)

Список результатов в виде таблички на каждый запрос с данными из хэша "form" пришедшим в ответ на запрос. (пример там же) И кнопка View для просмотра полного ответа. 

Желательно результат работы опубликовать на githube с промежуточными комитами.
